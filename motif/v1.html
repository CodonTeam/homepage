模型介绍

Codon Motif Vision 1 是一个实验性的视觉 Tokenizer (Visual Tokenizer) 模型，作为后续多模态模型的感知前段，它支持任意尺度的图像输入，并将其压缩为离散的 Latent Codes。

技术规格

量化 (Quantizer)

采用 LFQ (Lookup Free Quantization)，一种无查表的向量量化技术，计算包含熵 (Entropy) 和困惑度 (Perplexity)。

位置编码

集成 2D RoPE (Rotary Positional Embedding) 二维旋转位置编码，增强空间感知。

注意力机制

Encoder 和 Decoder 均使用 Spatial Multi-Head Attention (空间多头注意力)。

编码器 (Encoder)

结合 ConvBlock 和 ResBasicBlock，支持动态输入尺寸。

解码器 (Decoder)

使用 PixelShuffle 进行上采样重建。

默认配置

  • 输入/输出通道: 3 (RGB)
  • Patch Size: 16
  • Latent Dim: 256
  • Codebook Dim: 16
  • Encoder: Hidden Dim 256, 1 ResBlock, 4 Heads
  • Decoder: Hidden Dim 256, 3 ResBlocks, 4 Heads
  • RoPE Max Len: 4096

使用方法

1. 安装依赖

pip install orbit-torch

2. 模型加载

使用 load_pretrained 方法加载权重。

导入模块

from orbit.model.motif.vision.v1 import MotifV1

场景 A：加载全量模型

model = MotifV1() model.load_pretrained('motif-v1.safetensors')

场景 B：仅使用 Encoder (提取特征)

Quantizer 必须随 Encoder 加载以确保量化准确。

# 1. 实例化主模型 model = MotifV1() # 2. 从子模块加载权重 model.encoder.load_pretrained('motif-v1_encoder.safetensors') model.quantizer.load_pretrained('motif-v1_quantizer.safetensors') # 3. Use encode method # indices, mask, z_q = model.encode(x)
